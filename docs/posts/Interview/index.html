Build relationship.

Use patient name.

Explain why.

Collaborate.

Establish comfort.

Greet

"My name is MYNAME. I'm a TITLE. You can call me MYALIAS."

Frame

Structure the meeting. "I want to ask you some questions and do a physical exam."

Get consent. "Is that okay with you?"
